セットアップ／デプロイ／URL／バックエンドの「正」は引き続き `AGENTS.md`。

---

## 共有（Share）

- 投稿カード・投稿詳細・他ユーザープロフィール・マイページに**共有ボタン必須**。実装を消さない。
- 共有は `navigator.share`、不可時はリンクコピー。
- ディープリンク: 投稿 `?post=<id>`、プロフィール `?u=<user_id>`。起動時 `restoreDeepLinksFromUrl()` で復元。
- プロフィール再読み込みでホームに戻さない（`?u=` を維持）。

## PR / アフィリエイト送客

- **全部の投稿に運営の自動PRを出さない。** 必要なカテゴリだけ（`AFF_AUTO_CATS`）。
- **ご飯（food）・呑み（drink）は自動PRなし**（現地店舗は地図で十分。意味がない）。
- 旅行の「宿・ホテル」は**県だけの検索にしない**。投稿の**スポット名＋県**で楽天トラベル検索（その宿の予約に近づけたい）。
- 通販向きカテゴリは**投稿のスポット／ジャンル**を検索語に使う（定番ワードだけにしない）。
- 投稿者の `aff_url` があればそれだけ表示（運営リンクは出さない）。

## 地図リンク

- Yahoo!マップのキーワード検索は公式どおり **`/search?q=`**（`p=` は無効）。
- 座標がある投稿は Yahoo `/place?lat=&lon=` を優先。

## 運用・エージェント連携

- 変更の公開は `git push origin main` → GitHub Pages のみ（Netlify 禁止）。
- セットアップ／UI方針／バックエンドが変わったら **`AGENTS.md` を更新**し、ユーザーに **Claudeアプリ用コピペ共有文**を出す。
- 新しいプロダクト判断を会話で確定したら、**このファイルに追記**してから作業を終える。
